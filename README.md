# Customer Segmentation Analysis

## Project Overview

This project uses K-Means Clustering to segment customers based on Annual Income and Spending Score. The goal is to identify customer groups and generate business insights for targeted marketing.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Scaling
5. Elbow Method
6. K-Means Clustering
7. Cluster Visualization
8. Business Insights

## Results

Five customer segments were identified using K-Means Clustering. These segments help businesses understand customer behavior and create effective marketing strategies.
